# 지구온난화로부터 우리 도시를 지켜라

기후 정책 시뮬레이터 사용 설명서

---

## 시작하기

1. 선생님이 알려준 주소(또는 QR 코드)로 접속한다. 인터넷 없이 HTML 파일을 직접 열어도 실행된다.
2. 도시 이름을 입력하고 모둠원의 역할을 확인한다.
3. **시작** 버튼을 눌러 2030년 정책 회의를 시작한다.
4. 태블릿에서 화면이 작으면 오른쪽 위 **전체화면** 버튼을 사용할 수 있다.

---

## 모둠 역할

| 역할 | 담당 |
|------|------|
| **환경부 장관** | 환경·산림·탄소 관련 정책 승인 |
| **경제부 장관** | 모든 정책의 예산 승인, 대출 결정 |
| **과학기술부 장관** | 기술·탄소 제거 정책 승인 |
| **에너지·산업부 장관** | 에너지 전환 정책 승인 |

---

## 게임 목표

2100년에 두 가지 목표를 동시에 달성해야 한다.

- 🌡️ **기온 변화 2.0°C 이하**
- 🏙️ **도시 발전도 70점 이상**

---

## 화면 구성

### 왼쪽 상단 도시 그래픽
정책에 따라 도시 모습이 바뀐다. 나무가 늘거나, 매연이 줄거나, 창문이 어두워지는 등의 변화를 확인해 보자.

### 왼쪽 하단: 예측 결과
현재 정책대로 진행했을 때 2100년 기온·농도·발전도를 미리 보여준다.

### 오른쪽: 정책 패널
슬라이더를 올려 정책 강도(0~4단계)를 조절한다.

- **레벨 1~2**: 승인 없이 바로 적용
- **레벨 3~4**: 해당 부처 장관 모두의 승인 버튼이 필요하다
- 각 정책 카드에는 **도시 발전도**와 **예산 사용**이 함께 표시된다.

> 💡 **장관별 승인은 턴당 2회까지** 가능하다. 어떤 정책에 승인을 쓸지 모둠이 함께 결정해야 한다.
> 승인 횟수를 넘기면 버튼에 **승인 횟수 초과**가 표시된다.

## 정책 안내

| 정책 | 효과 | 비용 | 발전도 |
|------|------|------|--------|
| 화석연료 사용 줄이기 | 배출량 직접 감소 (효과 큼) | 레벨당 7 | -2 |
| 재생에너지 확대 | 전력 배출 감소 | 레벨당 9 | +5 |
| 에너지 효율 기술 향상 | 에너지 사용량 감소 | 레벨당 7 | +6 |
| 산림 보호·복원 | 탄소 흡수량 증가 | 레벨당 5 | +2 |
| 도시 에너지 소비 줄이기 | 배출량 감소 | 레벨당 4 | -4 |
| 탄소 제거 기술 투자 | 대기 중 탄소 제거 | 레벨당 10 | +3 |

## 예산 규칙

- 턴마다 기본 예산 **40**이 주어진다.
- 정책을 새로 올리는 단계는 레벨당 비용을 모두 낸다.
- 이전 턴부터 유지하는 단계는 매 턴 **25% 유지비**를 낸다.
  - 예: 레벨당 8인 정책을 4단계로 유지하면 다음 턴 유지비는 `4 × 8 × 0.25 = 8`
- 정책을 낮췄다가 다시 올리면, 다시 올린 단계는 새 투자비로 계산된다.
- 예산이 부족하면 **경제부 장관 승인**으로 다음 턴 예산을 당겨 쓸 수 있다.
  - 한 턴 대출 한도: **12**
  - 전체 대출 한도: **24**
  - 마지막 2080 턴에서는 대출 불가
- **정책 확정**을 누르면 확인창이 한 번 더 나온다. 확인창에서 다시 **정책 확정**을 눌러야 다음 턴으로 넘어간다.

## 이벤트

게임 중 두 번의 이벤트가 발생한다. 모둠이 함께 선택을 결정해야 한다.
이벤트 선택지에는 **도시 발전도**, **2100년 온실기체 농도 변화**, **예산 사용**이 함께 표시된다.

| 연도 | 이벤트 | 선택지 |
|------|--------|--------|
| **2040** | 전력 수요가 빠르게 늘고 있다 | 값싼 화석연료 발전 일부 유지 / 재생에너지와 효율 설비 확대 |
| **2060** | 도시 확장과 주변 숲 보호 | 개발 우선 / 숲 보호와 복원 지역 확보 |

### 2040 이벤트 선택의 의미

- **값싼 화석연료 발전 일부 유지**: 예산을 적게 쓰고 도시 발전도는 높지만, 2100년 온실기체 농도에는 불리하다.
- **재생에너지와 효율 설비 확대**: 예산 부담은 크지만, 2100년 온실기체 농도를 낮추는 데 유리하다.

### 2060 이벤트 선택의 의미

- **개발 우선**: 도시 발전도 확보에 유리하지만, 숲의 탄소 흡수 도움은 약해진다.
- **숲 보호와 복원 지역 확보**: 발전도 증가는 작지만, 탄소 흡수에 유리하다.

## 순서 요약

```
도시명 입력 → 2030 정책 결정 → 정책 확정 확인
→ 2040 이벤트 선택 → 정책 조정 → 정책 확정 확인
→ 2060 이벤트 선택 → 정책 조정 → 정책 확정 확인
→ 2080 정책 조정 → 정책 확정 확인
→ 2100 결과 보고서 확인 및 저장
```

## 보고서 저장

마지막 화면에서 세 가지 내용을 작성한다.

- 우리 도시의 핵심 전략
- 가장 효과적이었다고 본 과학적 정책과 이유
- 성장과 기후 대응을 함께 보며 고민한 선택

**보고서 반영** 버튼을 누르면 작성한 내용이 미리보기에 들어간다.
**PNG 보고서 저장** 버튼을 누르면 결과 보고서 이미지가 저장된다.
긴 문장을 작성하면 보고서 이미지의 세로 길이가 자동으로 늘어난다.

보고서 제목은 `도시명 2100년 기후 결과 보고서` 형식으로 표시된다.
