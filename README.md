# 지구온난화로부터 우리 도시를 지켜라

중학교 3학년 과학 수업의 모둠 활동용 기후 정책 시뮬레이터다.

## 실행

`index.html`을 태블릿 브라우저에서 연다. 최종 수업용 화면은 이 HTML 파일 하나에 들어 있으므로 인터넷 연결이 필요하지 않다.

학생 개인정보는 입력하지 않는다. 시작 화면에서 입력한 도시명만 최종 PNG 보고서에 들어간다.

## 권장 수업 흐름

1. 도시명 입력과 역할 확인: 2분
2. 2030 정책 시작: 3분
3. 2040 전력 수요 이벤트와 조정: 4분
4. 2060 숲 보호 이벤트와 조정: 4분
5. 2080 최종 조정: 3분
6. 2100 보고서 작성과 저장: 4분

## 예산 규칙

각 턴은 기본 예산 `40` 안에서 새 정책 투자와 이벤트 지출을 결정한다. 이미 올린 정책 수준은 다음 턴에 유지되지만 같은 투자비를 다시 내지는 않는다.

현재 턴 예산을 넘는 선택은 경제부 장관이 승인하면 다음 턴 예산을 당겨 쓸 수 있다. 한 턴 대출 한도는 `12`, 전체 대출 한도는 `24`이며 마지막 2080 턴에서는 다음 턴 예산을 빌릴 수 없다.

## 과학 모델 메모

이 도구는 En-ROADS에서 착안한 중학생용 축약 모델이다. 정책 선택이 온실가스 배출량, 대기 중 온실가스 농도, 기온 변화로 이어지는 흐름을 비교하도록 설계했다.

테스트용 계산 모듈은 `js/`와 `tests/`에 남아 있다. 실행 확인은 `npm test`로 한다.
